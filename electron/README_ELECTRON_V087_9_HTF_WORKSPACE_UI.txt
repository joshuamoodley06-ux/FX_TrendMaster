# FX TrendMaster Electron v087.9 — HTF Workspace UI

Patch focus: UI only.

Changes:
- Mark panel now has 3 internal modes at the top:
  - HTF Engine
  - Manual Events
  - Case Save
- HTF Engine mode is the default mapping workspace.
- Manual event taxonomy is hidden unless Manual Events mode is selected.
- Case save tools are available inside Mark panel under Case Save mode.
- NGMC dock is slimmer, taller, and runs down the right side so the chart/map stays visible on the left.
- Floating panel is narrower and taller to support map-left / tools-right workflow.
- Sticky HTF footer adds Reject All and Save State.

No backend changes in this patch. Use with backend v149.
