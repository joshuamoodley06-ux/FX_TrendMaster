v086.8 Selected Candle Range Anchors

- Added explicit Macro/Weekly/Daily range anchor buttons.
- Set M/W/D High uses the selected candle high.
- Set M/W/D Low uses the selected candle low.
- External/Extreme location markers no longer silently move the active fib range.
- Current high/low formation still can update the active range when marked as the new structure.
- Mark queue remains bundle based, so one candle can carry multiple facts.
- Backend unchanged.
