CHAINED STATE MACHINE IDEA
==================================================

A range breach + reclaim should cause cascading state transitions.

Example:
State 0:
Current Range High = 2350

State 1:
Price breaches high and stages new extreme 2380.

State 2:
Price closes back inside old range.
Engine:
- Old high becomes Ref H
- New staged high becomes current high
- Logs reclaim relationship
- Links retracement to old range

Abandoned vs reclaim:
Breach staged = price breaks active boundary.
Reclaim = later candle closes back inside old boundary.
Abandoned = price breaks boundary and continues forming structure outside old range without closing back inside within allowed confirmation window.
